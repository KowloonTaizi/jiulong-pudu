## 費用設定
開啟 `js/app.js`，找到：
`prices:{karma:0,creditor:0,ancestor:0,lineage:0,rice:0,offering:0}`

將 0 改為廟方正式單價。因尚未取得正式價格，目前顯示「金額請廟方確認」。
